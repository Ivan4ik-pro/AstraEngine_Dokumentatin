What is AstraEngine?

AstraEngine is a modular, data-driven game engine written in C++20 with a Lua scripting layer. It features a deferred PBR rendering pipeline, Bullet-powered physics, a native entity-component-system (ECS), spatial audio, and experimental multiplayer networking via WebSocket/UDP transport.

🎮

Game-Ready Architecture

Scene graph, asset pipeline, prefab system, and hot-reload support out of the box.

⚡

GPU-Accelerated

Vulkan & Metal backends with automatic batching, instancing, and LOD management.

🧩

ECS Core

Sparse-set entity-component-system for cache-friendly, ultra-fast game logic.

🌐

Cross-Platform

Windows, macOS, Linux, iOS, Android, and WebAssembly from a single codebase.

Quick Start

Clone the repository and build from source, or grab a prebuilt binary below. Here's your first scene in AstraScript (Lua):

AstraScript · Lua 
-- main.lua — minimal AstraEngine scene
local Engine = require("astra.core")
local Scene  = require("astra.scene")

-- Create a new scene
local scene = Scene.new("MyWorld")

-- Spawn a mesh entity with a rigid body
local box = scene:spawn({
  mesh     = "meshes/cube.aml",
  material = "materials/pbr_stone.mat",
  position = Engine.Vec3(0, 5, 0),
  physics  = { shape = "box", mass = 10.0 }
})

-- Register update callback
scene:on_update(function(dt)
  box:rotate(Engine.Vec3(0, dt * 45, 0))
end)

Engine.run(scene)
Note: AstraEngine uses a left-handed coordinate system. Y-axis points up. All angles are in degrees by default unless Engine.set_radians(true) is called.

Rendering Pipeline

AstraEngine uses a multi-pass deferred renderer with physically based shading (PBR). The pipeline supports HDR, temporal anti-aliasing (TAA), screen-space ambient occlusion (SSAO), and real-time global illumination via voxel cone tracing.

G-Buffer Pass

Albedo, normal, roughness, metallic, and emissive data packed into MRT targets.

Lighting Pass

Clustered forward+ lighting with shadow maps and real-time area lights.

Post Processing

Bloom, lens flare, chromatic aberration, vignette, LUT color grading.

Ray Traced Shadows

Optional DXR / Vulkan RT pass for ultra-accurate soft shadows. RTX only

GLSL · Vertex Shader 
// AstraEngine PBR Vertex Shader
#version 460 core
layout(location = 0) in vec3 a_position;
layout(location = 1) in vec3 a_normal;
layout(location = 2) in vec2 a_uv;

uniform mat4 u_mvp;
uniform mat3 u_normal_matrix;

out vec3 v_world_normal;
out vec2 v_uv;

void main() {
  v_world_normal = normalize(u_normal_matrix * a_normal);
  v_uv           = a_uv;
  gl_Position    = u_mvp * vec4(a_position, 1.0);
}
